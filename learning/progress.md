## セッションログ

### 2026-04-13 (Session 1)
**学習内容: Phase 0 — モーションの基礎概念**

- 3Dキャラクターは「ボーン階層 + メッシュ」で構成される
  - ボーン = 操り人形の骨。親ボーンを動かすと子ボーンも追従する
  - メッシュ = ボーンにくっついた見た目（このプロジェクトでは箱型）
  - 約50パーツ（指まで含む）が `BODY_PARTS` で定義されている

- モーション = 毎フレーム（60fps）ボーンの回転・移動を変化させること
  - 一般的なゲーム: Blender等で事前に作ったアニメーションを再生（キーフレーム方式）
  - このプロジェクト: コードで毎フレーム計算（プロシージャル方式）

- このプロジェクトの攻撃モーションの流れ:
  1. AIが攻撃種別・パワーを決定
  2. `createSwingMotion()` で手の軌道3点 + 体幹プリセットを生成
  3. 毎フレーム `updateSwingMotion(dt)` で progress 0→1 に進め、手の位置と体幹の動きを補間
  4. IKソルバーが手の目標位置から肩・肘の角度を逆算
  5. 関節クランプ・足接地処理の後、画面に描画

- 攻撃モーションは2フェーズで構成:
  - 振りかぶり(windup): progress 0→0.6、ease-inでゆっくり
  - 打撃(strike): progress 0.6→1.0、ease-outで素早く

- 調整の主な対象:
  - `swing-presets.ts` の BodyMotion値（体幹の動き）
  - SwingMotionの軌道パラメータ（手の位置）

**関連ファイル:**
